# SmartAir City

## Mô tả
SmartAir City là nền tảng IoT giám sát chất lượng không khí trong thành phố, bao gồm các cảm biến đo bụi mịn (PM2.5, PM10), CO₂, nhiệt độ và độ ẩm. Dữ liệu được thu thập và gửi tới API .NET 8 và hiển thị trên frontend ReactJS.

## Cấu trúc dự án
- **Backend**: API Web (ASP.NET 8) cho việc nhận dữ liệu và lưu trữ vào MongoDB.
- **Frontend**: Dashboard web hiển thị dữ liệu trên bản đồ (Leaflet.js) và biểu đồ (Chart.js).

## Các bước cài đặt
1. Clone repository này.
2. Cài đặt .NET SDK và MongoDB.
3. Cài đặt các phụ thuộc backend và frontend.
4. Chạy ứng dụng.
